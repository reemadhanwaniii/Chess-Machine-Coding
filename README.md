## Requirements

### Solution Requirements

- Do we need to store data in any kind of database or in-memory db is fine
  - In-memory database is fine

- Do we need to implement any kind of network API based interaction or console driven app is fine
 - Console App is fine

- Do we need to write only class structure or implement working code
  - As much as working code is possible is required


### Feature Requirements

- Here we need to implement a console based 2 player chess game
- All the basic rules of chess game should be kept in mind(not all the advanced rules are mandatory)
- Basic feature like multi piece support, user should be able to move piece here and there,attack piece etc.
- Good to have - 
  - for every piece if we can have their dedicated piece structure.
  - Game should be in a playable state i.e code must be in a running state
  - Support bot based player as well


### Non-Functional Requirements

- Code should be modular and extendable
- Code should support all the relevant OOPs and Solid Concept.
- Code should be in a Running state.
